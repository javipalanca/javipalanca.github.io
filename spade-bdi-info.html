SPADE BDI Plugin

Overview

SPADE BDI is a plugin for the SPADE 3 Multi-Agent System (MAS) platform that enables the creation of intelligent agents using the Belief-Desire-Intention (BDI) cognitive architecture. The plugin allows developers to create hybrid agents with sophisticated reasoning capabilities using the AgentSpeak programming language.

Purpose and BDI Architecture

The Belief-Desire-Intention (BDI) architecture was created to provide a theoretical and practical framework for building rational agents that can:

  • Reason about their environment - Through beliefs about the current state of the world
  • Make goal-oriented decisions - Using desires to represent what they want to achieve
  • Execute planned actions - Via intentions that represent committed courses of action

Core BDI Components

  • Beliefs: The agent's knowledge about the world, itself, and other agents
  • Desires/Goals: States or conditions that the agent aims to bring about
  • Intentions: Committed plans of action that the agent is currently pursuing

Key Features and Capabilities

  • Parse and execute AgentSpeak (ASL) files for agent behavior definition
  • Support for Agentspeak-like BDI behaviors with logic programming syntax
  • Dynamic belief management (set, get, remove beliefs)
  • Custom actions and functions integration
  • KQML performatives support (TELL, UNTELL, ACHIEVE) for agent communication
  • Plan execution with triggering events, contexts, and action bodies
  • Variable handling with uppercase naming conventions
  • Robust error handling and plan failure recovery
  • Message passing between agents
  • Recursive list handling and manipulation

AgentSpeak Language

AgentSpeak is a logic-based programming language specifically designed for creating intelligent agents in multi-agent systems. It provides:

  • Plan Structure: TriggeringEvent : Context <- Body
  • Internal Actions: Start with '.' (e.g., .print(), .send())
  • Goal Achievement: Marked with '!' (e.g., !find_shelter)
  • Belief Syntax: belief(arguments) format
  • Variable Handling: Uppercase variables with '?' operator for retrieval

Basic Usage Example

from spade_bdi.bdi import BDIAgent

# Create BDI agent with AgentSpeak file
a = BDIAgent("BasicAgent@server", password, "basic.asl")
a.start()

# Manage beliefs dynamically
a.bdi.set_belief("car", "blue", "big")
a.bdi.print_beliefs()
