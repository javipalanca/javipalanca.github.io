SPADE Bokeh Server Plugin

Interactive Visualization Integration for Multi-Agent Systems

Overview

The SPADE Bokeh Server Plugin is a specialized Python tool designed to integrate interactive Bokeh visualizations with the SPADE (Smart Python Agent Development Environment) framework. It enables developers to render dynamic, interactive plots directly within agent templates, bridging the gap between agent-based systems and modern web visualization.

Purpose and Motivation

The plugin was created to address the need for real-time, interactive data visualization in multi-agent systems. By integrating Bokeh's powerful plotting capabilities with SPADE's agent framework, developers can:

  • Monitor agent behaviors and interactions in real-time
  • Create dashboards for complex multi-agent system analysis
  • Visualize agent communications and state changes dynamically
  • Build web-based interfaces for agent system monitoring and control

Key Features

  • Seamless Integration: Easy integration with SPADE agent web services
  • Dynamic Plot Rendering: Real-time plot updates within agent templates
  • Web Controller Support: Built-in web controller functionality for plot management
  • Multiple Plot Support: Ability to add multiple interactive plots per agent
  • Bokeh Server Integration: Full access to Bokeh's interactive plotting capabilities
  • Template Integration: Simple JavaScript injection for Jinja2 templates

Basic Usage Example

import spade
import spade_bokeh
from bokeh.plotting import figure
from bokeh.layouts import column

class MyBokehAgent(spade_bokeh.BokehServerMixin, spade.agent.Agent):
    async def setup(self):
        self.web.start(port=10000)
        self.bokeh_server.start()
        self.bokeh_server.add_plot("/my_plot", self.create_plot)

    def create_plot(self, doc):
        # Create interactive Bokeh plot
        plot = figure(title="Agent Data Visualization")
        # Add plot elements here
        doc.add_root(plot)
